# Session 03 — source materials (2026-05-22)

Drop the raw materials from this session into the subfolders below. These aren't published to the Pages site — they're the source materials that the eventual `sessions/2026-05-22.html` page will draw from.

## Where things go

- **`raw-transcript/`** — the raw audio/text transcript of the session itself (the human conversation between Jake and Tanner). Any format is fine: `.txt`, `.md`, `.vtt`, `.srt`, etc.
- **`claude-code-transcript/`** — the Claude Code session transcript (the messages/tool calls from working with Claude). Export from Claude Code or paste as `.md`.
- **`visuals/`** — screenshots, diagrams, sketches, anything visual. Common formats: `.png`, `.jpg`, `.svg`, `.pdf`.

Anything else (audio clips, code snippets, links to share) can go in this folder at the root.

## Naming

No strict naming required — just keep it readable. e.g. `raw-transcript.md`, `claude-code-session.md`, `01-final-design.png`.
