How to read the stack

Bottom-up. Vercel is the platform — the ground everything deploys to. Node.js runs on top of it, providing the JavaScript runtime. TypeScript adds types; React adds components; Next.js wraps React into a full framework.

Tailwind and shadcn/ui sit at the top — they're what users actually see and interact with. Each layer depends on the ones below it.

Mental model — how data moves

Browser → Next.js → Agent → Supabase → Vercel
